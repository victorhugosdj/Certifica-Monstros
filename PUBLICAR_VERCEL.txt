## 🚀 PRÓXIMAS AÇÕES — Publicar em Vercel

### PASSO 1: Conectar Vercel com GitHub

1. Abrir: https://vercel.com/new
2. Clicar "Import Git Repository"
3. Buscar por: "Certifica-Monstros" ou seu repositório
4. Clicar "Import"

### PASSO 2: Configurar Projeto

Na tela de configuração do Vercel:

1. **Project Name:** certifica-monstros
2. **Framework Preset:** Other (ou deixar em branco)
3. **Root Directory:** ./frontend (IMPORTANTE!)
4. **Build Command:** deixar vazio
5. **Output Directory:** . (ponto)
6. **Environment Variables:** (próximo passo)

### PASSO 3: Adicionar Variáveis de Ambiente

Antes de fazer deploy, adicionar:

```
VITE_SUPABASE_URL = https://ibmembnxtbpsehqdorme.supabase.co
VITE_SUPABASE_KEY = eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
VITE_API_BASE_URL = http://localhost:8000 (ou seu backend)
```

### PASSO 4: Deploy

1. Clicar "Deploy"
2. Esperar 2-3 minutos
3. Vercel dará URL como: https://certifica-monstros.vercel.app

### PASSO 5: Testar

No site novo em Vercel:

```
1. Abrir: https://certifica-monstros.vercel.app
2. Fazer login
3. Ir para Dashboard
4. Verificar:
   ✓ Resumo ainda mostra
   ✓ Ranking aparece
   ✓ Desempenho por módulo funciona
   ✓ Sem erros de console (F12)
```
